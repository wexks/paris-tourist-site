# Туристический сайт города Париж

## Тема проекта
Туристический сайт города Париж

## Описание проекта
Многостраничный web-сайт, посвящённый городу Париж.  
Сайт содержит информацию о городе, достопримечательностях, событиях и контактные данные.  
Проект выполнен с использованием HTML5 и CSS3, с адаптивной версткой для мобильных устройств, планшетов и десктопов.

## Страницы сайта
- Главная — index.html
- О городе — pages/about.html
- Достопримечательности — pages/catalog.html
- События — pages/events.html
- Контакты — pages/contacts.html

## Используемые технологии
- HTML5
- CSS3 (Flexbox, Grid, адаптивная вёрстка)

## Особенности проекта
- Семантическая HTML-разметка
- Чистая структура CSS (reset.css, base.css, layout.css, components.css)
- Единый визуальный стиль для всех страниц
- Навигационное меню одинаково на всех страницах
- Адаптивность для разных устройств
